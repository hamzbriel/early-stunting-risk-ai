Tentang Proyek

Early Stunting Risk AI

Sistem prediksi risiko stunting end-to-end yang menggunakan Machine Learning untuk membantu tenaga kesehatan dalam melakukan deteksi dini dan intervensi pada anak-anak yang berisiko stunting.

Proyek ini dibangun sebagai demonstrasi lengkap dari pengembangan AI, mulai dari pembuatan dataset sintetis, pelatihan model, hingga deployment aplikasi web yang production-ready.

Teknologi

Machine Learning

  • • Scikit-learn, XGBoost, CatBoost
  • • SHAP (Explainable AI)
  • • Pandas, NumPy

Backend

  • • FastAPI
  • • Pydantic v2
  • • Uvicorn

Frontend

  • • HTML5, CSS3, JavaScript
  • • Tailwind CSS
  • • Jinja2 Templates

Deployment

  • • Render
  • • Git & GitHub

Fitur Utama

  • Prediksi risiko stunting dengan 20+ faktor kesehatan
  • Explainable AI dengan SHAP untuk transparansi hasil
  • Rekomendasi kesehatan yang dipersonalisasi
  • Interface modern dan responsive
  • REST API yang terdokumentasi lengkap
